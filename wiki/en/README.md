# Oihana PHP Auth — Documentation (English)

Welcome to the **`oihana/php-auth`** library documentation.

This library is a composable PHP toolkit to protect APIs:

- JWT verification against a remote JWKS (cached public keys),
- role-based access control (RBAC) with [Casbin](https://casbin.org/),
- fine-grained capabilities (`PARAM:…`, `FIELDS:…`) to restrict request parameters or returned fields,
- ready-to-use validation rules (`oihana\auth\rules`),
- PSR-15 friendly traits that plug into any Slim/Symfony stack.
